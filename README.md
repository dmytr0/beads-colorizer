# BeadsColorizer

Веб-застосунок для нумерації схем бісеру. Завантажте фото схеми — застосунок знайде всі кружечки-бісеринки, визначить унікальні кольори та проставить цифру всередині кожного кружечка.

![Favicon](static/favicon.svg)

## Можливості

- Автоматична детекція кружечків (OpenCV HoughCircles)
- Кластеризація кольорів у LAB-просторі з налаштовуваним порогом (threshold) — усуває артефакти JPEG
- Збільшення зображення до 2000px по мінімальній стороні (LANCZOS4)
- Цифри вписуються в кружечок і не виходять за межі
- Автоматичний контрастний колір цифр (формула WCAG)
- Ручний color picker для кожного кольору окремо з миттєвим live-preview
- Завантаження готового PNG у повній роздільній здатності

## Встановлення

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

```bash
python app.py
```

Відкрийте [http://127.0.0.1:5999](http://127.0.0.1:5999)

## Використання

1. Натисніть **Завантажити фото** і оберіть схему бісеру (JPG або PNG)
2. Налаштуйте **Threshold** (рекомендовано 20–40) — чим більше, тим менше унікальних кольорів
3. Натисніть **Обробити** — прогрес-бар покаже 4 кроки обробки
4. Після завершення зображення з цифрами з'явиться на сторінці
5. У легенді кольорів натисніть **A** поруч з будь-яким кольором, щоб змінити колір цифр через color picker
6. Подвійний клік на кольоровий квадрат — скидає на автоматичний контраст
7. Натисніть **Завантажити PNG** — збережеться файл у повній роздільній здатності

## Стек

| Компонент | Технологія |
|---|---|
| Веб-сервер | Flask |
| Детекція кіл | OpenCV HoughCircles |
| Обробка зображень | OpenCV + Pillow + numpy |
| Кластеризація кольорів | Жадібний алгоритм у LAB |
| Прогрес у реальному часі | Server-Sent Events (SSE) |
| Рендеринг цифр | HTML5 Canvas (клієнт) |
| Завантаження | `canvas.toBlob()` |
